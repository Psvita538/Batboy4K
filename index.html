Batboy4K Discord Bot : Your Versatile Discord Companion

Batboy4K is your all-in-one Discord bot, designed to bring fun and functionality to your server. Whether you want to liven up conversations, manage your community, or explore new servers, Batboy4K has you covered.

Fun and Games

  • +roll: Simulating the roll of a die between 1 - 12
  • +flip: Flips a coin which is heads and tails
  • +fortune: See what it grants you
  • +joke: Generate a joke

Moderation Tools

  • +ban [user] [reason]: Bans a user
  • +kick [user] [reason]: Kicks a user
  • +warn [user] [reason]: Warns a user with a DM
  • +purge: Deletes messages

Server Information and Management

  • +serverinfo: Display information about the server
  • +total_members: Fetches the members in total in this server
  • +invite: Gets the invite of servers

Utility

  • +timer [time]: Set a timer for a specified amount of time
  • +system: Checks the system CPU, Disk, and RAM
  • +ping: Shows the latency rate
  • +feedback [message]: Send feedback to the bot owner
  • +join: Joins the same voice chat you're in
